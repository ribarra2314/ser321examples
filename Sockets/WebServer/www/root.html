You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /guess -- uses two numbers to guess the random number use /guess?num1=[integer]&num2=[integer] where integer is between 1-10 inclusively Enter 11 for each integer to test if guess is correct
  • /distance -- uses two numbers to find the distance use /distance?num1=[integer]&num2=[integer]
